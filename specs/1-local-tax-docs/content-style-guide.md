# Content Style Guide: 개조식 공문서 스타일

**Feature**: 1-local-tax-docs
**Date**: 2026-01-28

## 개조식 문서 구조

모든 MDX 콘텐츠는 공무원이 익숙한 **개조식 공문서 형식**으로 작성한다.

### 기본 원칙

1. **서술형 금지**: 긴 문장의 서술형 설명 대신 개조식 항목으로 작성
2. **계층 구조**: 명확한 번호/기호 체계로 계층 표현
3. **간결성**: 한 항목당 한 가지 정보만 기술
4. **법령 용어**: 공식 법령 용어 사용

### 번호 체계

```
1. 대항목
   가. 중항목
      1) 소항목
         가) 세부항목
            (1) 세세부항목
```

### MDX 작성 예시

**❌ 잘못된 예 (서술형)**:
```mdx
# 취득세 세율

취득세는 부동산이나 차량 등의 재산을 취득할 때 납부하는 지방세입니다.
개인이 주택을 매매로 취득하는 경우에는 주택 수와 지역에 따라
다른 세율이 적용됩니다. 예를 들어 조정대상지역에서 1주택을 취득하면
1%에서 3%의 세율이 적용되고...
```

**✅ 올바른 예 (개조식)**:
```mdx
# 취득세 세율 - 매매 - 주택

## 1. 개요

가. 적용 대상: 매매로 주택을 취득하는 경우
나. 근거 법령: 지방세법 제11조
다. 시행일: 2024.01.01

## 2. 개인 납세자

### 가. 1주택

#### 1) 조정대상지역

| 취득가액 | 전용면적 | 취득세 | 지방교육세 | 농특세 | 합계 |
|---------|---------|--------|-----------|--------|------|
| 6억원 이하 | 85㎡ 이하 | 1.0% | 0.1% | - | 1.1% |
| 6억원 이하 | 85㎡ 초과 | 1.0% | 0.1% | 0.2% | 1.3% |
| 6억원 초과 9억원 이하 | 85㎡ 이하 | 1.0~3.0% | 0.1~0.3% | - | 1.1~3.3% |
| 9억원 초과 | - | 3.0% | 0.3% | 0.2% | 3.5% |

※ 세율 산식: (취득가액 × 2/3억원 - 3) / 100

#### 2) 비조정대상지역

| 취득가액 | 전용면적 | 취득세 | 지방교육세 | 농특세 | 합계 |
|---------|---------|--------|-----------|--------|------|
| 6억원 이하 | 85㎡ 이하 | 1.0% | 0.1% | - | 1.1% |
| ... | ... | ... | ... | ... | ... |

### 나. 2주택

#### 1) 조정대상지역

가) 세율: 8.0%
나) 지방교육세: 0.4%
다) 농특세: 0.6%
라) 합계: 9.0%

#### 2) 비조정대상지역

가) 세율: 1.0~3.0%
나) 비고: 1주택과 동일

### 다. 3주택 이상

(생략)

## 3. 법인 납세자

### 가. 일반 법인

1) 세율: 12.0%
2) 지방교육세: 0.4%
3) 농특세: 1.0%
4) 합계: 13.4%

### 나. 비영리법인

(생략)

## 4. 관련 규정

가. 지방세법 제11조 (취득세 세율)
나. 지방세법 시행령 제28조 (세율 적용 기준)
다. 지방세특례제한법 제XX조 (감면 규정)
```

## 콘텐츠 구성 요소

### 필수 섹션

1. **개요**: 적용 대상, 근거 법령, 시행일
2. **세율표**: 조건별 세율 (표 형식)
3. **관련 규정**: 법령 조항 목록

### 선택 섹션

- 주의사항 (※ 기호 사용)
- 계산 산식
- 참고사항

### 제외 항목

- ~~사례/예시~~ (마이그레이션 안 함)
- ~~Q&A~~
- ~~서술형 설명~~

## MDX 컴포넌트 사용

### 주의사항 표시

```mdx
<Alert type="warning">
※ 조정대상지역 여부는 취득일 기준으로 판단
</Alert>
```

### 세율표

기본 Markdown 표 사용:
```mdx
| 구분 | 세율 | 비고 |
|------|------|------|
| 1주택 | 1~3% | 가액별 차등 |
```

### 법령 인용

```mdx
> 지방세법 제11조 제1항
> 부동산을 취득한 자는 취득세를 납부하여야 한다.
```

- [ ] 세율표 포함
- [ ] 법령 근거 명시
- [ ] 사례/예시 제외
