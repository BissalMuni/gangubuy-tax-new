<h2 id="summary">
  <Outline level={1}>요약</Outline>
</h2>

이 리프는 지방세 취득세 관련 가산금·중가산금 및 납부지연가산세의 연혁적 변천(세율 변화)을 정리한다. 2024년 1월 1일부터 시행된 납부지연가산세 통합 체계(납부불성실가산세+가산금+중가산금)와 적용 요령을 설명하며, 취득세·교육세·농특세별 신고불성실·납부불성실·이자상당액 적용 여부를 표로 제시한다.


<h2 id="concepts">
  <Outline level={1}>주요 개념</Outline>
</h2>

- **가산금** — 납부기한 내 세금을 납부하지 않을 경우 최초 1회 부과되는 부가금으로, 2005년까지 5%, 2006년부터 3%로 인하되었다.
- **중가산금** — 가산금 부과 후에도 납부하지 않을 경우 매월 추가로 부과되는 부가금으로, 2018년까지 월 1.2%, 2019~2023년 0.75%, 2024년부터 0.66%로 단계적으로 인하되었다.
- **납부지연가산세** — 2024년 1월 1일 시행으로 기존의 납부불성실가산세·가산금·중가산금을 통합한 단일 가산세로, 납세의무 성립 시기에 따라 종전 또는 신규 제도가 적용된다.
- **신고불성실가산세** — 취득세 등 신고 시 무신고 또는 과소신고에 대해 부과되는 가산세로, 취득세에는 적용되나 교육세·농특세에는 적용되지 않는다.
- **납부불성실가산세** — 납부기한을 경과하여 세금을 납부하지 않는 경우 1일 22/100,000(2022.6.6.~) 또는 25/100,000(이전) 비율로 부과되는 가산세이다.
- **이자상당액** — 취득세 및 지방교육세·농특세 납부 지연 시 부과되는 이자 성격의 부담으로, 세목별 적용 여부가 다르다.

  <Outline level={1}>원문</Outline>
</h2>

<h2 id="section-4">
  <Outline level={1}>가산금 및 중가산금</Outline>
</h2>

```text
기    존           →            현 재 / 가 산 금 : 5%   ( ~ 2005년까지)  →      3%   ( 2006년부터 ~ ) / 중가산금 : 1.2%  ( ~ 2018년까지)  →    0.66% (2024년부터 ~ ) /  0.75% (2019~2023) / (60개월)   최대 72%             →      최대 39.6%        /  최대 45%
```



※ 취득세 신고시 가산세•이자상당액 등 납부대상

```text
세목 | 신고불성실 / (무신고,과소신고) | 납부불성실 | 이자상당액
취득세 | ○ | ○ 22/100000 (22.6.6 ~ ) / 25/100000 (19.1.1. ~) | ○ 22/100000 (24.1.1 ~ ) / 25/100000 (20.1.15. ~)
교육세 | × | ○ 22/100000 (22.6.6 ~ ) / 25/100000 (19.1.1. ~) | ×
농특세 | × | ○ 22/100000 (22.2.14 ~ ) / 25/100000 (19.2.11. ~) | ×
```



<h2 id="section-5">
  <Outline level={1}>납부지연가산세 (2024.1.1.시행)</Outline>
</h2>

```text
납부지연가산세 =  납부불성실가산세(1일 22/100000) + 가산금(1회 3%) / + 중가산금(매월 0.75%) / ※ 중가산금 매월 0.75% => 0.66%(2024년부터) / 적용요령 / - 2024.1.1. 전에 납세의무가 성립하는 경우 종전의 가산금제도 운영 / - 2024.1.1. 이후 납세의무가 성립하는 경우 새로운 납부지연가산세 적용
```
