# 세금 정보 시스템

한국의 각종 세금 정보를 **체계적으로 관리하고 조회**할 수 있는 웹 애플리케이션입니다.

## 주요 기능

### 취득세
부동산, 차량 등을 취득할 때 부과되는 지방세 정보를 제공합니다.
- 유상취득 (매매, 교환, 경매)
- 무상취득 (상속, 증여, 재산분할)
- 원시취득 (신축, 개축)

### 재산세
매년 6월 1일 기준 재산 소유자에게 부과되는 세금 정보를 제공합니다.
- 주택 재산세
- 토지 재산세
- 건축물 재산세

### 자동차세
자동차 소유에 대한 지방세 정보를 제공합니다 (연 2회 부과).
- 승용차
- 승합차
- 화물차
- 이륜차

### 검색
모든 세금 정보를 통합 검색할 수 있습니다.

---

## 사용 방법

1. **좌측 네비게이션**에서 원하는 세금 종류 선택
2. **세부 항목**을 클릭하여 상세 정보 확인
3. **검색 기능**으로 특정 정보 빠르게 찾기

---

## 법적 고지

이 시스템은 정보 제공 목적으로만 사용되며, 실제 세무 상담이나 법적 조언을 대체할 수 없습니다. 정확한 세무 정보는 전문가와 상담하시기 바랍니다.
