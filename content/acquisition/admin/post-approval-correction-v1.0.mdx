<h2 id="주의사항">
  <Outline level={1}>주의사항</Outline>
</h2>

<Callout type="caution">
- **결재 완료 후** 반드시 다시 해당 화면으로 진입하여 **[정정내역 대장반영]** 처리 필요
- 대장반영을 하지 않으면 정정 내용이 실제 대장에 반영되지 않음
- 처리 완료 후 민원인에게 안내할 것
</Callout>
