# 주택의 토지만 증여할 때

> 주택의 토지만 증여하는 경우 안분 계산식을 활용한 과세표준 산정 방법

  { id: "계산방법", label: "계산 방법" },
  { id: "입력방법", label: "입력 방법" },
]} />

---

<h2 id="계산방법">
  <Outline level={1}>계산 방법</Outline>
</h2>

주택의 토지만 증여하는 경우, 토지분에 해당하는 주택가격을 **안분 계산**하여 산정합니다.

<Outline level={2}>계산식</Outline>

```
주택가격(토지분) = 원래주택가격 × (토지가격 ÷ 전체시가표준액)
```

<h2 id="입력방법">
  <Outline level={1}>입력 방법</Outline>
</h2>

<Outline level={2}>처리 순서</Outline>

1. 기존 **주택가격을 지움**
2. 토지가격/전체시가표준액 비율로 **안분 계산**
3. 원래주택가격을 곱한 금액을 산출
4. 계산된 금액을 **수기로 주택가격에 입력**

<Callout type="caution">
- 시스템에서 자동 계산되지 않으므로 **반드시 수기 입력** 필요
- 안분 비율 계산시 소수점 이하 처리에 주의
</Callout>
