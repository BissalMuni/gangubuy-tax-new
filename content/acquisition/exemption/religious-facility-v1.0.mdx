# 16. 종교시설 면제

> 종교단체 부동산 취득 시 취득세 면제 요건, 구비서류 및 추징사유 안내.

---

## 1. 감면 내용

<Callout type="info">

**지특법 §50**: 취득세 **100% 면제**, 최저한세 미적용, 농특세 면제

- **재산세 및 지역자원시설세 면제**: 과세기준일 현재 해당 사업에 직접 사용
- 종교단체 또는 향교가 **제3자의 부동산을 무상으로 해당 사업에 사용**하는 경우를 포함

</Callout>

## 3. 취득세 추징사유

<Callout type="caution">

다음의 경우 감면받은 취득세가 **추징**됩니다:

1. 해당 부동산을 취득한 날부터 **5년 이내**에 수익사업에 사용하는 경우
2. 정당한 사유 없이 그 취득일부터 **3년이 경과**할 때까지 해당 용도로 직접 사용하지 아니하는 경우
3. 해당 용도로 직접 사용한 기간이 **2년 미만**인 상태에서 매각·증여하거나 다른 용도로 사용하는 경우

</Callout>
